# Apraksts
Uzrakstiet programmu C ++, kas ievada konsolē izveidojiet tic-tac-toe spēli. Spēlē ir jābūt 2 spēlētājiem.

- Ievadīt tekstu
- Apstrādāt datus
- Izmantojiet matemātiskās formulas
- Izmantojiet cikl "for"
